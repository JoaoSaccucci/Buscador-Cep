\# Buscador de CEP 🔍



Aplicação Java desenvolvida para realizar consultas de endereços de forma dinâmica utilizando a API \*\*ViaCEP\*\*.



\## 🚀 Funcionalidades

\* Consulta de endereços a partir de um número de CEP.

\* Integração com API REST utilizando `HttpClient`.

\* Manipulação e conversão de dados JSON para objetos Java com a biblioteca \*\*Gson\*\*.

\* Geração de arquivos `.json` com o resultado da busca.



\## 🛠️ Tecnologias Utilizadas

\* \*\*Java 17\*\*

\* \*\*Maven\*\* (Gerenciador de dependências)

\* \*\*Gson\*\* (Google JSON library)

\* \*\*ViaCEP API\*\*



\## 📦 Como executar

1\. Clone o repositório.

2\. Certifique-se de ter o Maven instalado.

3\. Execute a classe `Principal.java`.

4\. Insira o CEP desejado quando solicitado no console.
